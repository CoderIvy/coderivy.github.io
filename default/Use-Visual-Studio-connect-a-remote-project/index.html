Use Visual Studio connect a remote project

Description:
Use Visual Studio & server URL to connect all your remote projects from same server (the one you have access permission).

  1. Click ‘View’ -> ‘Team Explorer’


  2. Team Explorer shows as following picture
    click the ‘Manage Connection’ button (the green button) in the top


  3. Enter your server URL, click ‘Add’
    after a while, all projects from the server which you have permission will show as below


  4. Choose a project you want to connect and click ‘Connect’ button in the bottom

  5. The project’s full URL will show as below
    Choose a folder which you want to save the code and click ‘clone’ to clone the project from remote to local
