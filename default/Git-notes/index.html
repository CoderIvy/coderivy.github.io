Git notes


description:
These are my personal git notes for my daily life.

check file status <br><br>

git clone RepositoryURL``` clone a github repository to you local folder. 

add the file you name it. <br><br>

git add . ``` add all files you has modified. 

commit the changes you have added and add a comment for this change ad the same time. (**Note:** If you miss '-m', it will open a VIM file for you to add comment for this commit after this command) <br><br>

git pull``` pull code from remote repository (Note: to avoid overwrite code in remote repository, it is better to use ‘git pull’ before every ‘git push’ )

push your local changes to remote repository <br><br>

git stash``` save current changes to current branch

list your stash list in current branch<br><br>

git stash pop``` show the latest chage which you stashed before

drop the latest stash code which you stashed before<br><br>

git mergetool ``` open gitâ€™s merge tool to handle merge event

show all commit histories, including your local changes<br><br>

git config --global merge.tool meld``` set ‘Meld’ as default merge tool

set 'Meld' file path to default merge tool<br><br>

Git checkout -b NewBranchName RemoteBranchName``` create a new branch based on a remote branch

Delete the most recent commit, **keeping** the work you've done (**Note:** this command works for commits haven't push to remote repository, the number show how many local commit you want to reset)<br><br>

git reset --hard HEAD~1``` Delete the most recent commit, destroying the work you’ve done 

show all branches the repository has, including the branches you never used<br><br>

git checkout BranchName``` switch to named branch

delete named branch<br><br>

Git merge BranchName``` merge branch (Note the BranchName should be the one your want fetch data from and your current branch should be the one you want get more data)

Any feedback is welcome! :)
