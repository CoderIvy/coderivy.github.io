Three ways to break a long word in a small container 2019-06-10

Description:
I met a problem when I fetch data from the database for a table, there is an unusual long string text inside the data, and the table column has a relative width so the data just stretched the column. I found 3 ways to break an unusual long word in a relatively small container, there are,

 word-wrap:break-word;  word-break: break-all; & overflow-wrap: break-word;

Here is the sample code for these three properties,

<!DOCTYPE html>
<html>
<head>
<style>
p {
  width: 140px; 
  border: 1px solid blue;
 
}

p.a {
  word-wrap:break-word;
}

p.b {
  word-break: break-all;
}

p.c {
  overflow-wrap: break-word;
}
</style>
</head>
<body>

<h1>break an unusually long string at container width</h1>

<h2>default:</h2>
<p>Thisissomeveryveryverylong word. Words will break according to usual rules.</p>

<h2>word-wrap:break-word:</h2>
<p class="a">Thisissomeveryveryverylong word. This text will break at any character.</p>


<h2>word-break: break-all:</h2>
<p class="b">Thisissomeveryveryverylong word. This text will break at any character.</p>

<h2>overflow-wrap: break-word:</h2>
<p class="c">Thisissomeveryveryverylong word. This text will break at any character.</p>

</body>
</html>

How to make the native scrollbar thinner in Google Chrome 2019-05-06

Description:
Add a native scrollbar to the project and make the scrollbar thinner.

Most people knew how to add a scrollbar to your pages when the content of an element is too big to fit in the specified area. So, I will just give a brief introduction to CSS overflow property. overflow property has the following values:

  • visible - Default. The overflow is not clipped. The content renders outside the element’s box
  • hidden - The overflow is clipped, and the rest of the content will be invisible
  • scroll - The overflow is clipped, and a scrollbar is added to see the rest of the content
  • auto - Similar to scroll, but it adds scrollbars only when necessary

Please note that: the overflow property only works for block elements with a specified height.

First and the easiest way to set a scrollbar’s width is to use the scrollbar-width property. scrollbar-width is a property that controls the width or “thickness” of a scrollbar. scrollbar-width accepts the following values:

  • auto is the default value and will render the standard scrollbars for the user agent.
  • thin will tell the user agent to use thinner scrollbars, when applicable.
  • none will hide the scrollbar completely, without affecting the element’s scrollability.
  • <length> is being debated, but (if added) would be a maximum width of the scrollbar.
